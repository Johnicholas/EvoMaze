EvoMaze

This is the sample code given out as part of an Advanced Object-Oriented Programming lecture given at Imperial College in 2012. It's a small Java program that evolves two-dimensional grid-based maps. I've tried to comment it as much as possible to give an idea for what things can be tweaked. There's also some simple output code so you can watch the mazes evolve and improve over generations.
